Tabla en html

 

es una estructura que permite organizar y mostrar datos en filas y columnas. Se utiliza la etiqueta <table> para crear una tabla, y dentro de ella, se utilizan etiquetas adicionales para definir las filas y celdas.

Formulario en html

es una estructura que permite a los usuarios ingresar datos en una página web y enviarlos a un servidor para su procesamiento. Se define utilizando la etiqueta <form>, que puede contener varios elementos de entrada, como campos de texto, botones, menús desplegables y casillas de verificación. Estos elementos permiten recopilar información del usuario, como nombre, correo electrónico y preferencias.

Etiqueta table

 

La etiqueta <table> en HTML se utiliza para crear una tabla que organiza datos en filas y columnas.

Etiqueta from

La etiqueta <form> en HTML se utiliza para crear un formulario que permite a los usuarios ingresar datos y enviarlos a un servidor.

Atributos de table

 

Los atributos de <table> son propiedades que se pueden utilizar para definir el comportamiento y la apariencia de una tabla en HTML. Estos atributos permiten personalizar aspectos como el borde, el espaciado y el tamaño de la tabla, mejorando su presentación en la página web.

Etiqueta input

La etiqueta `<input>` en HTML se utiliza para crear campos de entrada en un formulario, permitiendo a los usuarios ingresar datos.

Etiqueta tr

 

La etiqueta <tr> en HTML se utiliza para definir una fila en una tabla.

Etiqueta label

La etiqueta <label> en HTML se utiliza para definir una etiqueta para un elemento de entrada en un formulario, mejorando la accesibilidad y la usabilidad al asociar texto descriptivo con el campo correspondiente.

 

Atributo tr

 

L os atributos de la etiqueta <tr> son propiedades que se pueden usar para definir características específicas de una fila en una tabla.

Etiqueta select

La etiqueta <select> en HTML se utiliza para crear un menú desplegable que permite a los usuarios elegir una opción de una lista de valores predefinidos.

Etiqueta td y th

 

La etiqueta <td> en HTML se utiliza para definir una celda de datos en una fila de una tabla.

 

La etiqueta <th> en HTML se utiliza para definir una celda de encabezado en una tabla, que generalmente se muestra en negrita y centrada.

Etiqueta textarea:

La etiqueta <textarea> en HTML se utiliza para crear un área de texto multilinea en un formulario, permitiendo a los usuarios ingresar texto extenso.

Atributos de td y th

 

Los atributos de <td> y <th> son propiedades que permiten personalizar celdas en una tabla. El atributo colspan especifica cuántas columnas abarca una celda, mientras que rowspan define cuántas filas abarca. El atributo headers asocia una celda con encabezados relacionados, mejorando la accesibilidad. Los atributos align y valign controlan la alineación horizontal y vertical del contenido. Los atributos class e id permiten aplicar estilos CSS y manipulación mediante JavaScript.

Etiqueta fieldset, legend

La etiqueta <fieldset> en HTML se utiliza para agrupar elementos relacionados dentro de un formulario, creando un contenedor visual que puede mejorar la organización y la legibilidad. La etiqueta <legend> se utiliza dentro de <fieldset> para proporcionar un título o descripción breve que identifica el grupo de campos contenidos en el fieldset.

Tablas avanzadas

 

Las tablas avanzadas en HTML permiten una presentación más compleja de datos. Pueden incluir encabezados múltiples mediante atributos colspan y rowspan, aplicar estilos personalizados con CSS, y adaptarse a diferentes tamaños de pantalla para una visualización responsiva. Además, pueden incluir interactividad mediante JavaScript, permitir datos anidados con subtablas y utilizar atributos ARIA para mejorar la accesibilidad. Estas características mejoran tanto la funcionalidad como la estética de las tablas en aplicaciones web.

Etiqueta meter

La etiqueta <meter> en HTML se utiliza para representar un valor escalar dentro de un rango conocido, como por ejemplo una medición. Se utiliza comúnmente para mostrar datos como progreso, calificaciones o niveles de consumo, y puede incluir atributos que especifican el valor mínimo, máximo y actual.

Etiqueta thead

 

La etiqueta <thead> en HTML se utiliza para agrupar el encabezado de una tabla, organizando las filas de encabezado en una sección específica. Esto mejora la accesibilidad y la semántica, permitiendo que los navegadores y lectores de pantalla identifiquen fácilmente la parte de la tabla que contiene los títulos de las columnas.

Etiqueta progress

La etiqueta <progress> en HTML se utiliza para representar el progreso de una tarea en curso, como la carga de un archivo o la finalización de una operación. Esta etiqueta muestra una barra de progreso que indica visualmente el avance de la tarea, y puede incluir atributos para definir el valor actual y el valor máximo.

Etiqueta tfoot

 

La etiqueta <tfoot> en HTML se utiliza para agrupar el pie de una tabla, permitiendo que se contengan filas que generalmente incluyen totales o resúmenes. Esta etiqueta ayuda a mejorar la estructura y la accesibilidad de la tabla, y se coloca después de <thead> y antes de <tbody>.

Etiqueta datalist

La etiqueta <datalist> en HTML se utiliza para proporcionar una lista de opciones predefinidas que un usuario puede seleccionar mientras ingresa datos en un campo de entrada. Se asocia con un elemento <input> mediante el atributo list, permitiendo que los usuarios vean sugerencias mientras escriben, mejorando la experiencia de entrada de datos.

Etiqueta tbody

 

La etiqueta <tbody> en HTML se utiliza para agrupar el cuerpo de una tabla, organizando las filas de datos que contienen la información principal. Esta etiqueta ayuda a estructurar la tabla de manera más clara y semántica, facilitando la manipulación y el estilo con CSS.

 

Etiqueta summary

La etiqueta <summary> en HTML se utiliza para proporcionar un resumen o título para un elemento <details>. Al hacer clic en el <summary>, el contenido dentro del <details> se muestra o se oculta, lo que permite a los usuarios expandir o contraer información adicional de manera interactiva.

 

 

 

Tema 2. Define los elementos que componen la estructuración de una página web:

 

  1. Contenedor (wrapper): Es un elemento que envuelve toda la estructura de la página, proporcionando un límite para el contenido y facilitando el estilo y la alineación.

 

2.  Cabecera (header): Es la sección superior de la página que generalmente incluye el título del sitio, el logotipo y, a menudo, el menú de navegación.

 

3.  Contenido (content): Es el área principal de la página donde se presenta la información principal, como texto, imágenes y otros elementos multimedia.

 

4.  Menú (menu): Es una lista de enlaces de navegación que permite a los usuarios acceder a diferentes secciones del sitio web.

 

5.  Pie (footer): Es la sección inferior de la página que suele contener información de contacto, derechos de autor y enlaces adicionales, como políticas de privacidad o términos de uso.

 

6.  Lateral (sidebar): Es una columna a un lado del contenido principal que puede contener enlaces, widgets, anuncios o información adicional relacionada con el contenido.

 

7.  Etiqueta div: Es un contenedor genérico en HTML que se utiliza para agrupar y organizar otros elementos de la página, facilitando el estilo y la manipulación con CSS y JavaScript.

 

 

Tema 3. Identifica y describe los elementos que componen la estructuración semántica de una página web.

 

Los elementos que componen la estructuración semántica de una página web son etiquetas HTML que describen el contenido de manera significativa, mejorando la accesibilidad y la comprensión del documento. 

 

1.  <header>: Representa la cabecera de un documento o una sección, y generalmente incluye el título, logotipo y navegación. Proporciona información sobre el contenido que sigue.

2.  <nav>: Define un bloque de navegación, que contiene enlaces a otras páginas o secciones dentro del mismo documento. Facilita la navegación en el sitio web.

3.  <main>: Indica el contenido principal de un documento, que es único y relevante para la página. Solo debe haber un <main> por página.

4.  <section>: Representa una sección temática dentro de un documento. Se utiliza para agrupar contenido relacionado y a menudo incluye un encabezado.

5.  <article>: Se utiliza para contener contenido independiente que tiene sentido por sí mismo, como un artículo de blog, una noticia o un post. Cada <article> debe ser capaz de ser distribuido de forma independiente.

6.  <aside>: Representa contenido adicional relacionado con el contenido principal, como notas, enlaces relacionados o anuncios. Se puede considerar como un lateral o barra lateral.

7.  <footer>: Define el pie de una página o sección, y generalmente incluye información sobre derechos de autor, enlaces adicionales o información de contacto.

8.  <figure> y <figcaption>: La etiqueta <figure> se utiliza para encapsular contenido multimedia como imágenes, diagramas o gráficos, mientras que <figcaption> proporciona una descripción o título para el contenido dentro de <figure>.

9.  <h1> a <h6>: Representan los encabezados de diferentes niveles, desde el más importante (<h1>) hasta el menos (<h6>). Se utilizan para estructurar el contenido y ayudar a los motores de búsqueda a comprender la jerarquía del documento.
